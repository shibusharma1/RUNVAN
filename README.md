# RUNVAN Website

A modern, responsive, and professional website developed for **RUNVAN (Returned United Nations Volunteers Association Nepal)**.

The platform showcases RUNVAN's mission, programs, leadership, publications, certifications, biographies, gallery, and community engagement initiatives through a clean and accessible user experience.

![License](https://img.shields.io/badge/license-MIT-green)
![Status](https://img.shields.io/badge/status-active-success)

---

## Overview

RUNVAN is a network of Returned United Nations Volunteers committed to promoting volunteerism, sustainable development, knowledge sharing, and community engagement in Nepal.

This website serves as the organization's digital presence, providing visitors with information about:

* Organization Overview
* Vision, Mission & Objectives
* Programs & Projects
* Publications & Reports
* Leadership Team
* Leadership Biographies
* Certifications & Recognitions
* Photo Gallery
* Testimonials
* Contact & Engagement Opportunities

## Installation

### Clone Repository

```bash
git clone https://github.com/shibusharma1/RUNVAN.git
```

### Navigate to Project

```bash
cd RUNVAN
```

### Run Locally

Using PHP:

```bash
php -S localhost:8000
```

Or place the project inside:

```bash
xampp/htdocs/
```

and access:

```text
http://localhost/RUNVAN
```
